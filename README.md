# Climate REU 2026

## Project

Characterizing geographic and atmospheric influences on growing season length in the eastern United States.

## Repository Structure

```
data/           <- Downloaded datasets (CSV files from project website)
scripts/        <- Python analysis scripts
reports/        <- Jupyter notebook reports
figures/        <- Figures for presentations/reports
```

Later on, we can add directories for "writeup", "presentation", etc.

## Getting Started

1. Install Python 
2. Create a virtual environment and install dependencies:
   ```
   python -m venv .venv
   source .venv/bin/activate      
   pip install -r requirements.txt
   ```
3. Download datasets from the project website into `data/`
4. Open `meeting_notes.ipynb` in Jupyter to get started

## Meeting Notes

Update "meeting_notes.ipynb" with results, figures, and other items to be discussed.
